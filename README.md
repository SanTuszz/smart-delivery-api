# 🚀 Smart Delivery API

API REST desenvolvida com Java e Spring Boot para gerenciamento de entregadores e pedidos com seleção inteligente baseada em localização.

## 🧠 Funcionalidades

- Cadastro de entregadores
- Listagem de entregadores
- Seleção automática do melhor entregador (baseado em distância)
- Criação de pedidos
- Listagem de pedidos

## 🛠️ Tecnologias

- Java 21
- Spring Boot
- Spring Data JPA
- Hibernate
- Banco H2 (em memória)
- Maven

## ▶️ Como rodar o projeto

```bash
mvn spring-boot:run
